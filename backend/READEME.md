# NextJs-AirbnbClone Backend

This is the backend service for the NextJs-AirbnbClone project. It provides the necessary APIs and database interactions for the application.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [API Endpoints](#api-endpoints)
- [Contributing](#contributing)
- [License](#license)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/NextJs-AirbnbClone.git
   ```
2. Navigate to the backend directory:
   ```bash
   cd NextJs-AirbnbClone/backend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

1. Start the development server:
   ```bash
   npm run dev
   ```
2. The backend server will be running at `http://localhost:5000`.
